Description

Let's start to create the Product banner for your landing page!

First, start by creating a div inside the main tag. This div will contain your banner; it should have an id of home. Inside this div, create another div and it should have an id of content.

The second div having an id of content contains a title, a paragraph, and an anchor tag. This anchor tag goes to or is hyperlinked to the product section of the page. You can add any text of your choice in the title, paragraph, and anchor tag. The paragraph has a font-size of 1.25rem and the font-weight is 300.

To stop overflowing the content of the div having an id of home, set the correct CSS property such that the overflow of the content is hidden and make sure that the text inside the div having an id of home is centered.

The anchor tag inside the div having an id of content should look like a button.

Add these styles to make it look like a button: display it as a block on the same line, the borders are solid lines with a color of your choice with 1px thickness. The border-radius is set to 0.25rem The size of the font should be 1rem, but remove the text-decoration so that it won't underline the text. Let's give some padding to it â€“ add padding on the top and bottom of 0.375rem, and padding on the left and right of 0.75rem. You can set the text of the color and the background color to whatever you like.

Objectives

In this stage, you should have created the following in your first landing webpage:

  • A new div having an id of home inside the main tag.
  • Another div of id content inside the above-created div having an id of home. To include the content there is a title, paragraph, and an anchor tag present in the div of id content which is hyperlinked to the product.
  • The content of the div having an id of home, if overflown, should be hidden and the texts should be center-aligned.
  • The anchor tag present inside the div having an id of content should look like a button and the CSS properties to apply the same are mentioned above.
