Description

Until now, the header and the main section of your landing page must be completed. It's time to create a footer for your landing page.

Start by adding a new div having an id of `contact` to the footer tag . Now, add one more div with an id of contact-col1 inside the div having an id of contact and this acts as a column which is present inside the outer div which acts as a row. This column div(div having an id of contact-col1) will contain the same navigation links and the logo image from the navbar.

So, the first link inside the div having an id of contact-col1, which goes to the home link, will contain the logo image with an id of footer_link_logo, width and height of 64, same as mentioned in the header section, and the other anchor tags will be grouped inside another container div and the ids of these anchor tags are mentioned below. After this container div there is a p tag containing the year 2022.

The ids of the anchor tags

  • footer_link_home
  • footer_link_product
  • footer_link_contact

These three anchor tags inside a div container must be hyperlinked to the home, product, and contact in the given order as it was done while creating the header section. The content of these anchor tags must be Home, Product, and Contact in the given order.

Now, let's apply the styles to your footer section.

The footer tag has a maximum width of 100%. The div having an id of contact should have the display value of flex and the items must be wrapped. The div having an id of contact-col1 should have a maximum width of 100%.

The example has extra paddings and margins, you can choose to add them.

Objectives

In this stage your website should have:

  • A new div with the id of contact in the footer tag;
  • A div container inside the contact div that has the same navigation links and logo image from the navbar;
  • The first link with the logo image inside with the conditions mentioned above;
  • Wrap other links inside the div;
  • The paragraph tag with the year;
  • All tags should follow their styles described above.
