Description

In this stage, you are going to add a contact form to your footer.

Start by adding a new div with the id of contact-col2 i.e. adding another column after the div having an id of contact-col1. This div or this column will have a form tag. It will collect the visitor's info.

The first child element inside the form tag is the heading of level 5. After this heading, there are two divs present.

The first div contains a label and an input tag with an id of input. Visitors will input their emails, so the label should say Email and the input type should also be email. You have to set the bottom margin of this div to 16px (either use combinators or give the div an id of your choice to apply the CSS).

The second div contains a label. Inside, there will be a checkbox input and a text. Also, donâ€™t forget the for attributes for labels. You have to set the bottom margin of this div to 16px (either use combinators or give the div an id of your choice to apply the CSS).

Both of the inputs should be required to fill in.

The last item inside a form is a submit button with any text. Now, let's add some styles to our footer page.

For the heading tag inside form tag, the font weight should be 500, the line height should be 1.2, the font size is 1.25rem, The color can be of your choice. The value of the display property of the label element of the email input tag is of block type.

The submit button should have a width of 100%, the display of it should be as a block on the same line, and the borders are solid lines with any color in 1px thickness. There should be no text decoration. The border-radius is set at 0.25rem . Set the font size at 1rem, add top and bottom paddings of 0.375rem; left and right paddings of 0.75rem. Set the mouse cursor as pointer. Set the color and background color as you like.

When submitted, if everything goes well, the page should be refreshed. If users fail to enter any values, the browser should notify them that the input is required.

Objectives

In this stage your website should have:

  • A new div inside the contact div with a form inside;
  • A heading tag inside the form;
  • A div container wrapping a label and an input for email;
  • A div container wrapping a label and a checkbox;
  • A submit button for the form;
  • The page should refresh after submitting the form;
  • All inputs are required before submitting;
  • All tags must have the styles described above.
