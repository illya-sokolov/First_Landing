Description

Our first landing page now has a basic layout. In this stage, you are going to add a Navbar to your webpage.

As you already know, the navbar is present inside the header tag. Let's start adding the nav tag inside the header tag. The navbar contains four anchor elements i.e. four hyperlinks and all four have different id attributes as given below:

  • link_logo
  • link_home
  • link_product
  • link_contact

The first link having the id as link_logo goes to the home section which has an id as home and you are going to create this section in the next stage. For now, link the first anchor tag to the home. This anchor tag also contains an image that will be the logo of your webpage. The image should have a fixed width of 64 and height of 64 . Don't forget to add the title and alt attributes to improve the accessibility of your webpage.

The second anchor tag having the id as link_home also goes to the home section. The third and fourth anchor tag having the id as link_product and link_contact goes to the product and contact section which will be created in the next stages. The texts for all these three anchor tags (second, third and fourth) must be Home, Product, and Contact respectively.

Let's set the maximum width of the nav tag to 100% and as you know to align the content of the nav in a horizontal direction you have to set the display of the nav tag to flex with the items wrapped.

For the navbar, you can add the paddings and margins to create the space you like. You can also change the color of the links and remove the text decoration if you want to as given in the example below.

Objectives

In this stage, you should have created the following in your first landing webpage:

  • The nav tag is present inside the header tag.
  • There should be four anchor tags with the above-mentioned IDs inside the nav tag.
  • There is an image inside the first anchor tag and the above-mentioned conditions should be fulfilled.
  • The first and second anchor tag should go to the home.
  • The third and fourth anchor tag should go to product and contact.
  • The texts of these anchor tags must be the same as mentioned above.
  • The nav tag should have a maximum width of 100% and all the CSS properties that are mentioned above related to the nav tag should be present.
