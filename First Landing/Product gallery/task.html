Description

In this stage, you are going to create a mini gallery that contains the images of the products listed on your landing page.

Let's first start by creating a new <div> having an id of `product` just after the div having an id of home. This div contains two more divs having an id of col-1 and col-2. You can think of the outer div as a container acting as a row and the other two divs that are direct children as columns.

Each of the divs having id of col-1 and col-2 contains a div and an image for the content. The divs present inside the div having an id of col-1 and col-2 has a second-level heading and a paragraph too. If these statements don't look promising you can refer to the below representation to get an actual flow of the layout.

Here is an example of the above-explained structure:

<!-- ROW -->
<div id="product">
  <!-- COLUMN -->
  <div id="col-1">
    <div> <!-- content --> </div>
    <!-- image -->
  </div>
  <!-- COLUMN -->
  <div id="col-2">
    <div> <!-- content --> </div>
    <!-- image -->
  </div>
</div>

You can increase the number of div rows, but the tests will only check for the first one.

Now, you are done with creating a basic layout for the Product gallery. Let's apply some CSS to style whatever you've created using HTML.

The display of the div having an id of product should be flex and set the width to 800px. The div tags having the id of col-1 and col-2 must have center-aligned texts and any overflows should be hidden. The paragraph tags just after the second-level headings should have the font-size of 1.25rem and the font-weight of 300.

The images inside the divs having an id of col-1 and col-2 should have a width of 296px and the border-radius is 21px on the top and right, 0px on the bottom and left. In addition, they should have the src, title and alt attributes with the values of your liking.

The example has extra paddings and margins, you can choose to add them.

Objectives

In this stage, you should have created the following in your first landing webpage:

  • A new div having an id of product in the main just after the div having an id of home.
  • There are two div containers of ids col-1 and col-2 inside the div having an id of product, and inside these divs there is another div with the h2 and p tags.
  • There is an image tag inside each of the divs having an id of col-1 and col-2.
  • Apart from the above, there are different CSS properties applied that are mentioned above.
